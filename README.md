# python-jokenpo-game
Jogo de Jokenpô (Pedra, Papel e Tesoura) desenvolvido em Python, com foco em manipulação de bibliotecas padrão (random, os, time) e experiência do usuário via terminal.

# 🎮 Jokenpô Interativo em Python

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

Este projeto é uma implementação clássica do jogo **Pedra, Papel e Tesoura**, desenvolvida para consolidar conhecimentos em lógica de programação, estruturas de decisão e interação com o sistema operacional.

## 🛠️ Funcionalidades e Diferenciais
- **IA do Oponente:** Utilização da biblioteca `random` para garantir escolhas imprevisíveis do computador.
- **Interface Limpa:** Uso da biblioteca `os` para realizar o "clear" no terminal após cada rodada, mantendo o ambiente organizado para o usuário.
- **Robustez de Dados:** Implementação de tratamento de erros (`try/except`) e loops de validação para garantir que o sistema não encerre com entradas inválidas.
- **Controle de Tempo:** Uso de `time.sleep` para dar ritmo ao jogo e permitir a leitura das mensagens pelo usuário.
